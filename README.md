# Audio-to-MIDI
Converts audio recordings into MIDI files using Python for music editing and transcription.
Technologies used are: Python, Basicpitch, ICASSP2020Model, WAV, pandas, CSV, Soundfile, Machine learning

Technologies Used	Features
Python 	Core language for building entire project.

Basic Pitch	pitch detection model used to convert audio into MIDI.
ICASSP 2022 Model	used for accurate musical note detection.
Sound file, WAV	reading WAV audio files and input format used.
MIDI	used to store musical note information digitally.
pandas	Used to convert detected note events into CSV format for analysis and visualization.
CSV	Used to store extracted musical note data.
File Handling Technology	Python Binary File I/O used to save output.
